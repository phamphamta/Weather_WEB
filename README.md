# Weather_WEB

The Weather_WEB project is a simple web application that provides weather information. This project is built using HTML, CSS, and JavaScript.

## Goals

- Provide weather information based on the user's location.
- A user-friendly and easy-to-use interface.

## Technologies Used

- **HTML**: Structure of the web page.
- **CSS**: Design of the user interface.
- **JavaScript**: Interaction and logic handling.

## Installation

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/phamphamta/Weather_WEB.git
Open the index.html file in your web browser to run the application.
How to Use
Enter the name of a city in the search box to receive weather information.
The application will display the current weather data for that city.
